hex2bin (line 41)

Function to decode a string containing hex digits ('hexits') to the original binary string representation. This is the inverse operation for the built-in bin2hex function.

Raises a LogicException if an invalid hex digit string is given.

  • return: The decoded binary string representation.

  • string $hex: The hex digit string to decode.
